Welcome to the 5EAPI

The 5eAPI aims to be a RESTful API endpoint for Dungeons & Dragons 5th Edition content. This is free to use and available to all.

This project is a labor of love from several D&D players and while it is still currently under construction we welcome any and all to use what is available! We are working to get more content added as quickly as we can. We appreciate your patience in this and if there are any suggestions or idea please contact us using the information at the bottom of the page.

You can test various endpoint's below to see what is available and what data is returned in your request. We also have some documentation available.

Try it Now!
