Resources

There are several different instances where an API response will contain information from another resource or several other resources. These should be represented as a CollectionResource or as a CommonResource.

A CollectionResource will contain information about an associated model, ie: A Race will have a CollectionResource of languages. This should contain the basic information that will be important for that response as well as a url to the associated information's endpoint for more detail.

A CommonResource will contain information that has a commonality between all resources. This typically will be a collection of links to every associated resource in the response.
