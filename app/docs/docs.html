Classes

GET v1/classes

              

            

            

              
The information returned while accessing the index of classes endpoint will respond with a collection of class options.  Each one contains several links for more information and a short description of the class.

              

                
                

                  Name
                  Description
                  Data Type
                

                
                
                

                  type
                  The name of the class resource
                  string
                

                

                  class_options
                  The characters subrace if applicable
                  string
                

                

                  {class name}
                  This is the name of the class with related information
                  object
                

                

                  url
                  A url for the class with more indepth information
                  integer
                

                

                  short_description
                  A simplified description of the class
                  string
                

                

                  levels
                  A url the collection of levels available to the class
                  string
                

                

                  subclass_index
                  A url to a list of all available subclasses
                  string
                

                

                  subclasses
                  The collection of subclasses available
                  object
                

                

                  {subclass name}
                  The link to information about the particluar subclass
                  sting
