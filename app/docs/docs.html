Introduction

There are a few things of note before we begin:

info This API is consumption-only, thus the only HTTP method we accept is a GET request.

info There is No Authentication at this time. This is free for public use and consumption, be aware that this could change in the future as our server loads increase

Wrappers

At this time we do not have any wrappers for our API. If you would like to create one we would love to see it and get it added here!


Resources

There are several different instances where an API response will contain information from another resource or several other resources. These should be represented as a CollectionResource or as a CommonResource.

A CollectionResource will contain information about an associated model, ie: A Race will have a CollectionResource of languages. This should contain the basic information that will be important for that response as well as a url to the associated information's endpoint for more detail.

A CommonResource will contain information that has a commonality between all resources. This typically will be a collection of links to every associated resource in the response.


Races

The worlds of Dungeons & Dragons have many different races to choose from, each with their own unique characteristics and traits. There are several endpoint options to choose from in order to get race information from the API.
